# Ledgerscope.GeographicLocations

A library for getting at a snapshot in time of the geo locations info in Windows, and making it available to our dev machines build servers (including Linux ones).  

## Location


`ledgerscope/Utilities/Repos/Files/Ledgerscope.GeographicLocations`

## Build and deploy

The pipelines task builds it into a nuget package and deploys it to our nuget feed (LSFeed).
